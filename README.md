# DnD-Web-App

Overview
This is a DnD character creator made to keep track of a users
character stats, details, items, session notes and more.

Setup
Download the .zip file, and extract it to a location you want.
Open the index.html

This project features the ability to: 
-Keep track of main ability scores
-Keep track of character skill profiencies and modifiers
-Add attacks/spells...etc
  -Includes ability to write descriptions
  -Modify the damage dice that will be used
-Remove attacks if no longer needed
-Keep track of inventory items, values, and whether the item is equipped or not.
-Remove items if no longer needed
-Keep track of character backstory, sessions notes, or dm notes for the player
